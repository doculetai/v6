## tRPC procedure needed

In src/server/routers/student.ts (or a shared auth router), add:

```typescript
createProfile: publicProcedure
  .input(z.object({
    userId: z.string().uuid(),
    role: z.enum(['student', 'sponsor', 'university', 'admin', 'agent', 'partner']),
  }))
  .mutation(async ({ ctx, input }) => {
    await ctx.db.insert(profiles).values({
      userId: input.userId,
      role: input.role,
    });
  }),
```
