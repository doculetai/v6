Build the public certificate verification page at src/app/certificate/[token]/page.tsx (Server Component, NO auth required).

## Page Purpose
Anyone (university admissions, employers, visa officers) can verify a student's proof-of-funds certificate using a unique token. No login required.

## URL
/certificate/[token] where token is a UUID generated when the certificate is issued.

## Visual Design
- White card on dark navy background (even in light mode — looks official)
- Green "VERIFIED" badge top-left with shield/lock icon
- Doculet.ai logo top-right (text logo, no image dependency)
- All data in a clean label/value grid
- Bottom: security disclaimer text

## Invalid Certificate State
If token not found or certificate revoked:
```
❌  Certificate Not Found

This certificate could not be verified. It may have been
revoked or the link may be invalid.

If you believe this is an error, contact:
support@doculet.ai
```

## Metadata (for sharing)
```typescript
export async function generateMetadata({ params }: Props): Promise<Metadata> {
  // Server-side fetch to get certificate data
  const cert = await getCertificateByToken(params.token);
  return {
    title: cert ? `Proof of Funds — ${cert.studentName} | doculet.ai` : 'Certificate Not Found | doculet.ai',
    description: cert ? `Verified proof of funds certificate for ${cert.studentName}` : 'Certificate not found',
    robots: 'noindex', // Don't index individual certificate pages
  };
}
```
