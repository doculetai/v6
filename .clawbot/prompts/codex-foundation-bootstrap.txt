## Files to create

### src/db/index.ts
Drizzle client singleton using Supabase connection string from env:
```typescript
import { drizzle } from 'drizzle-orm/postgres-js';
import postgres from 'postgres';
import * as schema from './schema';

const connectionString = process.env.DATABASE_URL!;
const client = postgres(connectionString, { prepare: false });
export const db = drizzle(client, { schema });
export type DrizzleDB = typeof db;
```

### src/server/trpc.ts
tRPC init with Supabase auth context:
- `createTRPCRouter`
- `publicProcedure`
- `protectedProcedure` — requires valid Supabase session, throws UNAUTHORIZED if missing
- `roleProcedure(role)` — requires session + specific role from profiles table

### src/server/context.ts
tRPC context factory:
- Creates Supabase server client
- Gets session from Supabase
- Returns: { db, supabase, session, user }
