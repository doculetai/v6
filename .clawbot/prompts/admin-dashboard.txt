## PRODUCT CONTEXT

Doculet.ai is a Nigerian bank statement verification platform. The admin dashboard is the nerve center of the platform. Admins review documents, verify identities, detect fraud, issue certificates, manage users across all 6 roles, and monitor system health (Supabase, Dojah, Mono, Paystack, Resend integrations).

The admin's primary workflow is the review queue: items arrive when students upload documents or complete identity verification. Review types: Identity Verification, Bank Verification, Document Review, Sponsor Attestation, Certificate Generation. Each item has a priority (Normal 48h SLA, High 24h, Urgent 4h) and can be assigned to specific reviewers. The evidence viewer shows 4 panels: Bank Evidence (account name, balance ₦/USD, transaction history), Identity Evidence (BVN/NIN verification, name match score), Document Evidence (file preview, OCR results, previous versions), Sponsor Evidence (sponsor KYC status, pledge amount).

**Cross-persona handoffs**: Every admin decision creates a notification for the affected student. Approve document → student sees "Approved" green badge + trust score updates. Reject → student sees rejection reason + "Re-upload" button. Flag fraud → account suspended + student notified. Issue certificate → student gets "Certificate Ready" notification + can share. V5 gap: notifications weren't dispatched on admin decisions. All amounts in ₦, PII access must be logged in audit_logs. Copy from `src/config/copy/admin.ts`.

## Page Purpose
Platform admins monitor overall health: student volumes, document review queue, revenue, and system health.

## Page Layout

### Row 1: Platform Metrics (4 stat cards)
- Active Students: count + trend (↑12% this week)
- Pending Document Reviews: count + "X awaiting >24h" warning
- Monthly Revenue (₦): total disbursements processed this month
- Certificate Issued: count this month

Each stat card: large number + label + trend indicator + sparkline (mini chart, 7 days).

### Row 2: Two columns
**Left (2/3 width): Recent Activity Feed**
Chronological list of platform events (newest first):
- 🟢 Student verified: Kemi Adesanya (Tier 3)
- 📄 Document submitted: Chukwuemeka O. (passport)
- 💰 Disbursement: ₦200,000 → Adaeze N. from FundEd Corp
- 📧 Invite sent: olumide@gmail.com invited by student
- ✅ Certificate issued: CERT-2026-KA-001
- 🔴 Webhook failure: Paystack callback (retry 2/3)

Each event: icon + message + timestamp (relative) + link to relevant entity.
Infinite scroll or "Load More" button (20 per load).

**Right (1/3 width): System Health**
Status indicators:
- Sentry Error Rate: X errors/hour → green/amber/red
- Webhook Delivery Rate: X% success → progress bar
- DB Latency: Xms average → green (<100ms), amber (<500ms), red (>500ms)
- Storage Usage: X GB / Y GB
- Auth Success Rate: X%

Each indicator auto-refreshes every 60 seconds (polling).

### Row 3: Quick Actions
Buttons/links:
- Review Document Queue → /dashboard/admin/documents
- Manage Users → /dashboard/admin/users
- View All Certificates → /dashboard/admin/certificates
- System Settings → /dashboard/admin/settings
