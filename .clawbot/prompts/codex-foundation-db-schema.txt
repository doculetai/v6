## Schema files to create

### src/db/schema/users.ts
Tables: `users`, `profiles`

```typescript
// users — mirrors Supabase auth.users (subset of columns we need)
export const users = pgTable('users', {
  id: uuid('id').primaryKey(), // matches auth.users.id
  email: text('email').notNull().unique(),
  createdAt: timestamp('created_at').defaultNow().notNull(),
});

// profiles — role + onboarding state for every user
export const profiles = pgTable('profiles', {
  id: uuid('id').primaryKey().defaultRandom(),
  userId: uuid('user_id').references(() => users.id, { onDelete: 'cascade' }).notNull().unique(),
  role: text('role', { enum: ['student', 'sponsor', 'university', 'admin', 'agent', 'partner'] }).notNull(),
  onboardingComplete: boolean('onboarding_complete').default(false).notNull(),
  createdAt: timestamp('created_at').defaultNow().notNull(),
  updatedAt: timestamp('updated_at').defaultNow().notNull(),
});

// profileRelations
export const profileRelations = relations(profiles, ({ one }) => ({
  user: one(users, { fields: [profiles.userId], references: [users.id] }),
}));
```

### src/db/schema/students.ts

Key columns:

## Rules

- Use `drizzle-orm/pg-core`: `pgTable`, `uuid`, `text`, `boolean`, `timestamp`, `integer`, `jsonb`
- All monetary amounts: INTEGER (kobo/pence), never float or decimal
- All timestamps: `timestamp('...').defaultNow().notNull()` for createdAt, `.defaultNow()` for updatedAt
- All FKs: explicit `references(() => table.id, { onDelete: 'cascade' })`
- All enums: `text('col', { enum: [...] })` — no Postgres enum types
- Relations declared with `relations()` from drizzle-orm in the same file as the table
- No `any` types
- TypeScript strict mode
